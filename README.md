## ✅ Exercícios Desenvolvidos

### 1️⃣ Ex1MostrarNumeros

Exibe números conforme a proposta do exercício, praticando estruturas de repetição e saída de dados.

### 2️⃣ Ex2VetorInverso

Realiza leitura de vetor e exibe os valores em ordem inversa.

### 3️⃣ Ex3MediaNotas

Calcula a média de notas, reforçando operações matemáticas e manipulação de dados.

### 4️⃣ Ex4Consoantes

Identifica e conta consoantes em um conjunto de caracteres.

### 5️⃣ Ex5ParImpar

Classifica valores em pares e ímpares.

### 6️⃣ Ex6MediaAlunos

Calcula médias de alunos e auxilia na análise de desempenho.

### 7️⃣ Ex7VetorMath

Aplica funções da classe `Math` em operações com vetores.

## ▶️ Como Executar

### Pré-requisitos

- Java JDK instalado (recomendado Java 17+)
- Terminal (PowerShell, CMD ou terminal da IDE)

### Compilar

```bash
javac Exercicios/*.java
```

### Executar (exemplo)

```bash
java Exercicios.Ex7VetorMath
```

> 💡 Se sua IDE estiver configurada sem pacote, execute diretamente pela própria IDE (botão Run) para cada arquivo.
